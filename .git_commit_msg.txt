feat: 新增表单填写Agent(Ethan)及多项功能增强

## 新增功能
- 新增 Ethan (form-filler) Agent：信息采集与文档填写专家
  - 支持从 .docx/.xlsx/.pdf/.txt/.md 等13种格式文档中提取待填项
  - 对话式逐字段收集用户输入，支持AI自动生成字段内容
  - 字段选择界面(FieldSelector)：用户可勾选需要填写的字段
  - 文件选择界面(FileSelector)：多文档时让用户选择目标文档
  - 表单填写视图(FormFillView)：填写完成后预览并保存文档
  - 表单填写会话锁：填写过程中所有消息直接路由给Ethan
  - 自然语言意图分类：LLM理解用户意图（填写/跳过/AI生成/下一个等）
- 新增文档解析统一入口(docParser.ts)：支持多格式文档文本提取
- 新增 .docx 文件处理器(docxHandler.ts)：基于JSZip的XML解析与填写
- 新增 .xlsx 文件处理器(xlsxHandler.ts)：基于SheetJS的Excel处理
- 新增跨会话记忆系统(memoryStore)：记住重要信息、回忆历史分析
- 新增 Token 用量统计(tokenUsageStore)：追踪每次LLM调用的token消耗
- 新增新建对话按钮：支持清空当前对话重新开始
- 新增 Agent 对话面板右侧展示：子Agent工作过程可视化
